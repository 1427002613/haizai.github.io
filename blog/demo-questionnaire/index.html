demo——vue问卷调查

发表于 2016-10-06   |   分类于 demo

  • demo地址：vue问卷调查

  • 源码：github源码

  • 技术栈：

  1. 框架：vue，vuex, vue-router
  2. 样式：sass
  3. 图表UI：ichart
  4. ES6：babel
  5. 打包工具：webpack

  • 特点：

  1. 集成现在个人所会的技术栈，具体构造见最后的结构图。
  2. vue-router控制路由，共有问卷列表，填写问卷，问卷详情，新建问卷，4个路由页面。
  3. vuex控制数据流向，有两个具体修改数据的事件，创建新建问卷和提交答卷。数据存储在localStorage。
  4. 尝试了sass的$变量，@import引用，和一个简单却高效的mixin。

  • 感想与总结：

  1. 花了一周多时间完成，把现在会的都用上了~
  2. sass样式分成多个文件，在main.scss，把一个个都@import进来，结构更清晰了。
  3. 组件components 和 路由views分开，便于复用。
  4. 最后踩了个坑，JSON.stringify()会将Date转化为字符串，相当于调用Date.prototype.toISOString()。然而JSON.parse()并不会解码这个字符串，需要单独把每一个字符串调用new Date()。如果直接把Date储存过于复杂，我的解决方法是储存数据前在Date前加+，将其直接变为Number来储存，这样就不会受JSON.stringify()和JSON.parse()影响，需要转化回来时再使用new Date()。
