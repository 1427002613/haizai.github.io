demo——vuex实现简单todos

发表于 2016-09-28   |   分类于 demo

  • demo地址：vuex-todos

  • 源码：github源码

  • 技术栈：vue，vuex，babel，webpack

  • 特点：

  1. webpack打包，ES6用babel转译ES6，vuex来控制数据。
  2. 是之前的vue-todo的改进版，不再使用Bootstrap。
  3. 单击todo可以编辑它的内容。
  4. 数据存储在localStorage。

  • 感想与总结：

  1. 配置package和webpack了很久，终于可以简单的使用了。
  2. ES6语法起来一气呵成，箭头函数非常“优雅”。
  3. vuex控制数据，结构清晰，不再需要组件$dispath与$broadcast。
  4. Bootstrap限制太多，自己写样式开心多了~~

  • 结构：

.
├── webpack.config.js .... webpack配置文件
├── package.json ......... npm配置文件
├── index.html ........... html主入口
├── .babelrc ............. babelrc配置文件
└── src
    ├── App.vue .......... vue入口
    ├── main.js .......... webpack入口
    ├── components
    |   ├── foot.vue ..... 尾部组件
    |   ├── todo.vue ..... 每一个todo的组件
    |   └── todos.vue .... 全部todo和foot的组件
    ├── css
    |   └── todos.css .... 样式
    └── vuex   
        ├── actions.js ... vuex事件
        └── store.js ..... vuex仓库
