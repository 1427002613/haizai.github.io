demo——vue入门todos

发表于 2016-09-21   |   分类于 demo

  • demo地址：vue入门todos

  • 源码：github源码

  • 依赖：vue，Bootstrap

  • 特点：

  1. 简单的入门todos，vue + Bootstrap实现。
  2. 点击输入框左侧图标可以全选，点击左下侧×可以删除被选择的，点击右下侧课切换filter。
  3. localStorage中读取和存放数据，刷新后数据仍然存在。
  4. vue实现简单的过渡动画。
  5. vuex-todos后来用vuex写的改进版。

  • 感想与总结：

  1. 第一次用vue，熟悉了vue的各种选项。如（watch，computed，filters，methods，transition）
  2. localStorage储存时需要JSON.stringify()，localStorage读取时需要JSON.parse()。
  3. vue用起来真心好顺畅，再也不用操纵DOM，MVVM时代~~
