Bootstrap的sass源码学习笔记（二）：排版篇（上）

发表于 2016-09-17   |   分类于 Bootstrap   |  

可以结合官网起步排版一起学习
其中排版在bootstrap/type中，变量在bootstrap/variables中，（变量以//注释的方式显示）

标题

可以看出<h1>-<h6>与.h1-.h6的共同样式：font-family与color为继承，font-weight为500，line-height为1.1。
他们的子元素<small>或.small：font-weight减小为normal(400)，line-height减小为1，颜色变成固定的gray-light(#777)

其中$line-height-computed // floor(($font-size-base * $line-height-base)) // floor(14px * 1.428571429) // 20px
可以看出前3个margin-top为20px，后3个margin-top为10px。而margin-bottom均为10px。
前3个的small的font-size为自身的65%, 前3个的small的font-size为自身的75%。
不在h下的small的font-size为父级的85%

这是每个h的font-size，其中h5是基础的14px

页面主体

<p>的margin-bottom设为10px，而font-family;font-size;line-height;color;均继承于body。

中心内容

.lead, 值得注意的是这里出现的@media。font-size在大于768px时为21px, 小于768px为16px
