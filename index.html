demo——vue+jQueryAjax实现天气预报

发表于 2016-09-23   |   分类于 demo

  • demo地址：vue+jQueryAjax实现天气预报

  • 源码：github源码

  • 依赖：vue，jQuery，和风天气API

  • 特点：

  1. vue + jQuery的ajax实现简单的天气预报。
  2. 初始化时发送ajax获得本机ip的城市，再发送ajax返回天气预报。
  3. 城市下方的「点击切换」点击以后，可以输入国内其他城市名切换城市发送ajax。
  4. 「点击切换」输入内容以后出现浮出框供选择。
  5. css过渡虽然看似很简单，实际上发送ajax和css过渡异步进行，css过渡显示和ajax返回不堵塞。
  6. 写了一点响应式，使用flex布局，IE10+。

  • 感想与总结：

  1. 第一次用jQuery的ajax，稍微了解了一定的ajax。
  2. vue和ajax结合，途中也遇到了很多坑，最后用vue的created和transition解决了。
  3. 一共用3个ajax，流程是这样的：
    1. 初次打开时直接发送ajax返回本机ip。
    2. ip为参数发送ajax返回所在城市名。
    3. 城市名为参数发送ajax返回天气预报。
    4. 切换城市后，再回到第3步。
  4. ajax的第4步和css过渡异步进行，过程是这样：
    1. 切换城市时同时发送ajax和css过渡隐藏。
    2. 若ajax先返回，返回值赋值给newData。当css过渡隐藏结束时将newData赋给data，再css过渡显示。
    3. 当css过渡隐藏先结束，则什么都不干/(ㄒoㄒ)/~~，待到ajax返回时，返回值直接赋值给data，再css过渡显示。
    4. 这样以来无论过渡时间设置为多久，都不会出现BUG（别问我怎么知道了，调试了好久~~~）

demo——jQuery实现日历UI

发表于 2016-09-21   |   分类于 demo

  • demo地址：jQuery实现日历UI

  • 源码：github源码

  • 要求：百度ife2016任务41

  • 特点：

  1. jQuery实现，没有使用现成库或框架。
  2. 使用了javascript的构造函数模式及原型模式，只用了function Calendar(),Calendar.prototype={},new Calendar(),没有暴露其他变量。
  3. 上个月最后几天和下个月前几天计算得到，所以每月5行或6行不固定。
  4. 点击某天则active某天，上个月最后几天或下个月前几天，则先翻到那个月，再active。

  • 感想与总结：

  1. 第一次用jQuery完整的写一个组件，问题还是很多啦~~
  2. 整个流程：先init(),init最后执行render(),render最后绑定click()事件,click中lastMouth()和nextMouth()时再绑定render()。
  3. 计算日期过程调试非常多才成功，具体实现是这样：根据上个月最后一天的周几确定上个月还有几天，根据这个最后一天确定这个月，根据这个月最后一天的周几确定下个月初有几天，最后把这三部分合在在一起，每次render()时执行。
  4. 循环绑定耦合度高，不易维护。越发感觉到MVVM的好~
